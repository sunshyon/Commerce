# 技术栈
## 前端 
vue.js + 静态页面 + nginx静态服务器  
## 后端 
主体框架：.Net5微服务架构 + 前后分离 + 动态API + Mysql读写分离 + EFCore  
网关/授权：Ocelot + JWT  
Nosql：Redis(热点数据,幂等性) + ES(搜索引擎，日志记录)  
消息队列：RabbitMQ(异步写库)  
分布式日志：log4net + kafka + ELK
